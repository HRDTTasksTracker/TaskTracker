Inaccessible Webpage Quiz

Overview

This webpage has several accessibility errors. I am limiting the errors to those you've seen so far in the class.

Allowed Resources

  1. You can use any course and online resources as long as you work alone.

  2. Accessibility trumps other aspects of the presentation. So, your accessible version's presentation might be different from the original, but it should be structurally and semantically equivalent.

  3. Hint: The activities we did in class and material from the textbooks is sufficient reference material. To find this information on Canvas, click here.

What to do

  1. Resolve the accessibility issues

    1. There are eight accessibility issues.

    2. Rubric:

      1. You start with 2 points for attempting to resolve the issues regardless of success.

      2. For each issue

        1. You get one point for completely resolving the issue.

        2. You get half a point for partially resolving the issue. For example fixing it in one place but not in other places. Not all issues can be partially resolved..

      3. Total of 10 points

  2. Submit the updated HTML file.

    1. You may also submit CSS files if you do the challenge, but you can also use internal CSS.

  3. As a challenge, use CSS to make the accessible html render more like the original.
